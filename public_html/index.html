LINUX

pwd - shows present working directory
ls - lists files in the current directory
cd - changes directory (use cd .. to go up a directory)
mkdir - makes a directory
mv - renames files & directories
rm - removes/deletes files (use -r to remove folder with recursion)
touch - creates an empty file
vi - opens vi editor to edit files

GIT

git init - initializes a git repository
git add - adds all files to track for commit("." includes all files)
git commit -m - commits a unit of work, a complete set of changes
git checkout (-b) - switches to a branch(-b makes new branch and switches to it)
git push - pushes local commits to remote repository on github

DOCKER

docker ps - lists the running containers
docker kill - shuts the container off
docker compose up - Builds, (re)creates, starts, and attaches to containers for a service
docker compose down - Stops containers and removes containers, networks, volumes, and images created by up
