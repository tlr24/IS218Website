LINUX

pwd - shows present working directory
ls - lists files in the current directory
cd - changes directory (use cd .. to go up a directory)

Example:
Current directory: /home
cd student
New directory: /home/student

mkdir - makes a directory

Example: mkdir test
Creates test directory in current directory

mv - renames files & directories

Example: mv index.htm index.html
Renames the index.htm file to index.html

rm - removes/deletes files (use -r to remove folder with recursion)

Example: rm file1
Removes file1 from current directory

touch - creates an empty file

Example: touch file1
Creates a file named file1

vi - opens vi editor to edit files

GIT

git init - initializes a git repository
git add < filename > - adds all files to track for commit ("." includes all files)
git commit -m < message describing commit > - commits a unit of work, a complete set of changes
git checkout (-b) < branch > - switches to a branch (-b makes new branch and switches to it)

Example: git checkout -b newBranch
Creates and switches to newBranch branch

git push < remote > < branch > - pushes local commits to remote repository on github

DOCKER

docker ps - lists the running containers
docker kill < container-id > - shuts the container off

Example: docker kill 179d9f6e2b12
It will shut off the container with the container id 179d9f6e2b12

s docker compose up - Builds, (re)creates, starts, and attaches to containers for a service
docker compose down - Stops containers and removes containers, networks, volumes, and images created by up
